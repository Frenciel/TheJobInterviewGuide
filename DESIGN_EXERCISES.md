## Amazon Dashboard

- Define Objective/Requeriments
  * Create a Search textbox for products
- Define Limitations
  * Ranking system
  * Number of products (database)
  * Number of categories (database)
  * Number of write Transactions per month
  * Number or read Transactions per month
- Out of Scope
  * Suggestions
  * Infinite scrolling
- Actions/Actors
  * Users (JWT roles)
